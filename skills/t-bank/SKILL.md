# Т-Банк: только чтение

## Подготовка

1. Получить точную цель: счёт или карта, нужные данные и диапазон дат.
2. Выполнить `python3 scripts/integration_preflight.py t-bank`.
3. Предпочитать отдельный браузерный профиль либо существующую авторизованную Chrome-сессию.
4. Проверить точные origin `https://id.tbank.ru/` и `https://www.tbank.ru/` перед вводом или чтением данных.

## macOS и Windows

- На macOS рекомендовать FileVault и macOS Keychain; отдельный зашифрованный контейнер является дополнительной, а не обязательной возможностью.
- На Windows рекомендовать BitLocker и Windows Credential Manager; не заявлять дополнительное шифрование, если BitLocker не подтверждён.
- При отсутствии подтверждённого шифрования использовать ручной интерактивный вход без сохранения банковского пароля или TOTP навыком.

## Авторизация

- Первый вход, пароль, TOTP, SMS/push, CAPTCHA и подтверждение устройства выполняет владелец.
- Не читать cookies, local storage, токены или файлы браузерного профиля.
- Не использовать внутренние банковские API и не воспроизводить запросы страницы.

## Разрешено

- Читать только указанный баланс, операции в согласованном диапазоне и детали выбранной операции.
- Скачать явно запрошенную официальную выписку или справку.
- Сохранить минимальный результат в профильной задаче по выбранной владельцем модели приватности.

## Запрещено

Не выполнять платежи, переводы, обмен валюты, инвестиции, изменение лимитов и реквизитов, выпуск/закрытие продуктов, обращения за кредитом, автоплатежи и другие мутации. Не просматривать остальные продукты «заодно».

После чтения закрыть открытый агентом браузер без выхода из аккаунта, если владелец выбрал сохранение сессии. Не сохранять широкую банковскую историю в логах.
