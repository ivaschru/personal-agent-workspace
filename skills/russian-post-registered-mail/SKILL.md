# Электронное заказное письмо

## Подготовить комплект

1. Прочитать связанную задачу и определить адресата, точный адрес, назначение и обязательные документы.
2. Проверить PDF-файлы, страницы, читаемость, размер и порядок.
3. Если комплект нужно разделить, явно обозначить каждую часть и общий состав.
4. Выполнить `python3 scripts/integration_preflight.py russian-post-registered-mail`.

## Войти и заполнить

- Открыть `https://zakaznoe.pochta.ru/` через существующую авторизованную сессию либо использовать `gosuslugi` для входа через ЕСИА.
- Передать владельцу проверки личности и неожиданный выбор профиля.
- Сопоставить адрес по отдельным элементам и загрузить только утверждённые файлы.

## Подтверждение

Перед оплатой и отправкой показать одним блоком: отправителя, адресата, адрес, назначение, файлы, страницы и стоимость. Изменение любого элемента отменяет прежнее подтверждение.

- Ручной ввод карты, 3-D Secure, SMS/push и банковское подтверждение выполняет владелец.
- Финальную отправку выполнять только по прямому подтверждению точного резюме.

## Доказательства

После успешной отправки сохранить трек-номер, стоимость, дату, копию письма, квитанцию и подтверждение приёма. Открыть файлы и проверить адресата, дату и ненулевой размер. Зафиксировать следующий срок контроля доставки.
